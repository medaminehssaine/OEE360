Introduction

TS_OEE360 is a Python-based industrial simulation project that models Overall Equipment Effectiveness (OEE) using a discrete-event simulation framework with SimPy. It outputs synthetic time series data that can later be used for forecasting.

What is OEE?

Overall Equipment Effectiveness (OEE) is a standard metric used in manufacturing to measure productivity. It combines three components: Availability, Performance, and Quality.

Installation

You can install the required dependencies using pip:

pip install -r requirements.txt

Alternatively, use the provided batch file:

run_me.bat

Usage

TS_OEE360 provides a comprehensive framework for simulating and analyzing OEE data.

Running Simulation

To run the simulation and generate OEE data:

python scripts/run_simulation.py

This will generate a dataset in data/ultra_complex_OEE.csv containing time series data of OEE metrics.

Visualization

To visualize the OEE components:

from ts_oee360.utils import *
df = load_simulation_data()
plot_oee_components(df)

Forecasting

The generated data can be used for time series forecasting:

# Example of using the data for ARIMA forecasting
import pandas as pd
from statsmodels.tsa.arima.model import ARIMA

df = pd.read_csv('data/ultra_complex_OEE.csv', index_col='timestamp', parse_dates=True)
model = ARIMA(df['OEE'], order=(5,1,0))
model_fit = model.fit()
forecast = model_fit.forecast(steps=24)  # 24-hour forecast

Project Structure

The project is organized as follows:

ts_oee360/         # Main module
  ├── simulator.py     # SimPy classes (Machine, Supplier, PowerGrid)
  ├── generator.py     # Simulation runner
  └── utils.py         # Plotting & helpers
scripts/
  └── run_simulation.py
notebooks/
  └── exploration.ipynb
data/
  └── ultra_complex_OEE.csv  # Generated simulation data

API Reference

Detailed API documentation for the main components:

Simulator

Contains the SimPy simulation classes for modeling industrial equipment.

Generator

Includes the main simulation loop, data generation, and CSV writing functions.

Utils

Provides helper functions for data loading, cleaning, and visualization.
